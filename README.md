在製作此前測之前完全沒有使用過Tailwind與pug，在安裝這兩個套件的時候花了一些時間尋找安裝後能使用的套件與參考步驟，最後才發現上述參考資料中滿足需求的資料。

在使用Tailwind與pug上，畢竟沒有使用經驗，所以只好開著參考網頁，一有問題就馬上查，慢慢的完成切版。
